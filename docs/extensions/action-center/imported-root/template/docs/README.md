# Documentation Hub

This is a PowerToys Command Palette extension. It runs as an out-of-process COM server registered with PowerToys and provides commands and/or dock bands to the Command Palette.

This directory is the authoritative documentation source.

## Read Order

1. `CONTEXT.md` — why this extension exists and what it does. Read this first if you are new.
2. `ARCHITECTURE.md` — SDK primitives, file structure, and design patterns.
3. `RUNBOOK.md` — exact commands to build, deploy, reload, and debug.
4. `CONVENTIONS.md` — coding, naming, and docs standards.
5. `BUGS.md` — active issue ledger and known risks.
6. `ROADMAP.md` — planned features and milestones.

## Authority Levels

- Canonical: `ARCHITECTURE.md`, `RUNBOOK.md`, `CONVENTIONS.md`
- Operational: `BUGS.md`, `ROADMAP.md`
- Orientation: `CONTEXT.md` (background and motivation — not current truth for code shape)
